# Interfaz con BorderPane en JavaFX

## Descripción

Este proyecto consiste en desarrollar una aplicación básica en **JavaFX** que utiliza un **BorderPane** para organizar varios botones en las distintas regiones de la ventana: norte, sur, este, oeste y centro.

El objetivo del ejercicio es aprender a utilizar el layout `BorderPane` para estructurar interfaces de forma ordenada y dividida en zonas.

## Objetivos

- Crear una interfaz gráfica con JavaFX.
- Utilizar el layout `BorderPane`.
- Colocar un botón en cada región del contenedor.
- Comprender la distribución en norte, sur, este, oeste y centro.
- Practicar el diseño de interfaces estructuradas.

## Funcionamiento

1. Se inicia la aplicación.
2. Se crea una ventana principal.
3. Se utiliza un `BorderPane` como contenedor principal.
4. Se colocan botones en cada región:
   - Superior (Norte)
   - Inferior (Sur)
   - Izquierda (Oeste)
   - Derecha (Este)
   - Centro
5. La interfaz se muestra organizada por zonas.

## Ejemplo de interfaz

```text
+----------------------------------+
|            [NORTE]              |
| [OESTE]     [CENTRO]   [ESTE]   |
|            [SUR]                |
+----------------------------------+
```
